# WhatsApp-Instagram-Chat-Analysis
A data analysis project that visualizes WhatsApp and Instagram chat insights like message count, emoji usage, and active time patterns.
# Tools & Technologies used
Python :– For data analysis and backend processing.

Pandas & NumPy:– To clean and analyze chat data.

Matplotlib / Seaborn / Plotly :– For visualizing data (graphs & charts).

Streamlit / Flask :– To create a simple web interface (if it’s a web app).

Regex (Regular Expressions) :– To extract date, time, sender, and message from chat text files.

JSON / CSV :– For storing structured chat data .

HTML, CSS (optional) :– If custom frontend design is added .
